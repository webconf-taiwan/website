常見問題

活動相關

WebConf Taiwan 是個什麼樣的組織?

WebConf Taiwan 並不是一個正式的組織或企業，它是一個單純的「集結、分享」的理念，加上幾位熱情的志工們組成的團隊。我們的目的很簡單，就是「技術交流」、「交朋友」、「找人才、找工作」。希望可以透過這樣的機會，把同樣從事 Web 開發相關工作的朋友聚在一起。如果您跟我們有相同的理念，歡迎加入我們，為台灣 Web 繼續注入熱情。

網站上看到的那幾隻動物代表什麼意思?

這些動物分別代表一種程式語言、社群或生態，例如大象表示 PHP、駱駝表示 Perl、蛇表示 Python，紅寶石代表 Ruby, 畫筆代表設計師，蝴蝶代表某知名企業，而主角蜘蛛則是用網子 (Web) 把這些角色全部網在一起，表示 Web 的兼容並蓄，也是 WebConf Taiwan 的主要精神及活動宗旨。

有官方的社群帳號或 IRC 頻道嗎?

WebConf Taiwan 有以下幾個社群帳號：
  • Facebook：https://www.facebook.com/WebConfTaiwan
  • Twitter：https://twitter.com/WebConfTaiwan
  • Twitter：https://twitter.com/WebConfTaiwan
  • IRC: irc.freenode.net#webconftw
  • Plurk Channel：http://www.plurk.com/topics/WebConf
如果您在 Twitter 上發文的時候，建議您可用 #webconftw 或 #webconftw2013 做為本次活動的 hash tag。

議程相關

選了 Designer 相關的主題，可以到時候再換去聽 Developer 的演講嗎?

其實並沒有限定大家只能選擇 Designer 或 Developer 的主題，兩天的議程您可以自由選擇您喜歡的題目前往參加。

會有現場錄影或轉播嗎?

本次活動礙於硬體限制，不提供現場即時轉播，但所有演講均會有錄影，影片跟投影片在講師的同意後，在活動結束後便會開始進行剪輯並上傳至 Youtube 頻道。

會場相關

有規定每個人坐在固定的座位嗎?

沒有，只要保留第一排的座位給講師及工作人員即可。

會場有網路可以用嗎?

有的，會場備有免費無線網路可使用。活動現場每個座位也都有獨立的電源跟網路插座，雖然無線網路很方便，但仍建議可以自己帶實體網路線，穩定度會較無線網路來得好。

那我可以用自己的 3G 手機上網嗎?

使用個人行動網路是沒問題的，但為了避免現場太多的訊號干擾，我們強烈建議您儘量避免將個人的 3G 網路設定成基地台使用。

會場有提供開水或飲料嗎?

有的，在會議室兩側會全天候提供飲料，在下午時間亦備有點心供大家取用。

食物及飲料帶請勿帶進會議室!
